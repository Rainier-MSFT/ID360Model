Purpose

This is a test project created to troubleshoot and understand routing behavior in Azure Static Web Apps with integrated Azure Functions.

Architecture

  • Frontend: Static HTML/CSS/JavaScript hosted in Azure SWA
  • Backend: Azure Functions (PowerShell) for API endpoints
  • Routing: Configured via staticwebapp.config.json

Test Scenarios

  • Simple GET requests to API endpoints
  • POST requests with JSON bodies
  • Query parameter handling
  • Route parameter handling
  • Error responses and status codes
